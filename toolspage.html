Tools yang Digunakan


Adapun tools yang harus dipersiapkan oleh teman-teman untuk mendukung pembelajaran kita yaitu :
1. Visual Studio Code digunakan sebagai code editor/tempat untuk menulis, menyusun dan menjalankan kode JavaScript yang dibuat oleh teman-teman.
Alternatif : teman-teman juga bisa menggunakan Programiz dan Playcode sebagai code editor, dapat diakses secara dengan koneksi internet dan tidak perlu diinstall terlebih dahulu.
2. Browser (Google Chrome/Microsoft Edge/Safari/dan lain-lain) sebagai tempat untuk menampilkan output scripting/hasil dari kode JavaScript yang dijalankan di code editor.
3. NodeJS digunakan untuk menjalankan kode JavaScript yang sudah dibuat pada terminal di aplikasi code editor (Visual Studio Code)

Dan adapun extension dari Visual Studio Code yang harus dipersiapkan oleh teman-teman untuk mendukung pembelajaran kita yaitu :
1. Live Server digunakan untuk menjalankan server lokal untuk proses debugging tampilan dan desain HTML. Live server ini akan mendeteksi perubahan di dalam file HTML, CSS, dan JS dan melakukan live reload secara otomatis.
2. JavaScript (ES6) Code Snippets digunakan untuk kerangka snippets penulisan JavaScript dengan basis standar ES2015.
