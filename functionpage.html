Function


Fungsi/Function dalam JavaScript adalah kumpulan pernyataan yang dirancang untuk melakukan tugas tertentu atau menghitung nilai tertentu. Dengan menggunakan function, kita dapat mengelompokkan code menjadi bagian-bagian yang dapat digunakan ulang tanpa batas. Function itu sama definisinya dengan variable dan array. Jika variable adalah wadah yang dapat menampung 1 nilai, lalu array dapat menampung banyak nilai, dan function dapat menampung beberapa variable dan/atau array menjadi sebuah wadah.

function wadah(){
const nama "Budi"
const umur = 30

const mahasiswa = ['mahasiswal', 'mahasiswa2', 'mahasiswa3'];

const keluarga = {
ayah: "Budi",
ibu: "Siti",
anak: ['udin', 'udan', 'udon', 'eden']
}

console.log(nama)
} wadah() // output: Budi

Dari contoh diatas dapat kita lihat bahwa terdapat sebuah function bernama “wadah”, yang didalamnya terdapat beberapa variable, array dan object. Untuk menggunakan function tersebut kita hanya perlu memanggil nama dari function tersebut “wadah()”. Jangan lupa untuk menggunakan buka-tutup kurung “()” pada akhir dari nama agar function tersebut dapat langsung digunakan. Silahkan dicoba apa perbedaan ketika teman-teman memanggil sebuah function dengan cara tanpa tutup kurung “wadah” dan dengan tutup kurung “wadah()”.

Function dapat dipanggil berkali-kali sesuai kebutuhan kita. Itulah mengapa materi function ini wajib teman-teman kuasai agar memudahkan dalam membuat codingan.

Jenis-jenis Function

  • Function Declaration

    • Dikenal juga sebagai function statements adalah function yang telah dideklarasikan sebelumnya dan dieksekusi jika function tersebut dipanggil (call). Function declarations harus dimulai dengan kata function.

    function salam() {
    alert("Hello World!");
    }

    salam() //output : Hello World!

  • Function Expression

    • Function yang disimpan ke dalam suatu variable. Setelah disimpan, maka variabel tersebut dapat digunakan sebagai function. Function yang disimpan di dalam function expression dapat diberi nama atau bisa disebut Named Function Expressions (NFEs) ataupun tidak diberi nama sama sekali (anonymous).

    const halo = () => {
    console.log("Hello World!");
    }

    halo() //output : Hello World!

    atau bisa juga dengan contoh seperti ini :

    const tambah = function(a, b) {
    return a + b;
    };

    // Memanggil Function Expression
    const hasilTambah = tambah(5, 3);
    console.log(hasilTambah); // Output: 8

  • Arrow Function

    • Adalah sintaks JavaScript modern (ES6), Dengan arrow function kita bisa menulis function tanpa harus menggunakan kata kunci “function”, “return”, dan kurung kurawal. Dengan kata lain, membuatnya menjadi lebih singkat dan ringkas.

    const greetings = (name) => console.log('Hello, ' + name + '!');

    greetings("Lorem Ipsum") //output : Hello, Lorem Ipsum!

    Catatan : Arrow Function hanya bisa digunakan 1 baris. Maksudnya, di kodingan atas terdapat 3 baris, jika teman-teman melanjutkan kodingan “const greetings = (name) => ….” Hingga ke baris ke- dua maka kodingan tersebut akan error atau tidak akan terbaca karena aturan arrow function hanya 1 baris

    Parameter dan Argument

    Parameter adalah variabel yang ditentukan di definisi function, sedangkan argumen adalah nilai yang dikirim saat function dipanggil.

    function greetings(nama){ //nama = parameter
    console.log("Halo" + nama + "!");
    }

    greetings("Lorem Ipsum") //Lorem Ipsum = argument

    Pada contoh diatas dapat teman-teman lihat bahwa kita memiliki sebuah function dengan nama “Greetings” dan terdapat sebuah parameter yang bernama “nama”. Ketika kita ingin memanggil function tersebut maka kita harus mengirimkan argument ke function tersebut agar “console.log” dapat berjalan sebagaimana semestinya.

    function tambah(a,b){ //a,b = parameter
    console.log(a+b)
    }

    tambah(5, 10) //5,10 = argument
    //output : 15

    atau dengan contoh lain seperti berikut :

    function javascript(jam){ //jam = parameter
    jam = "10.00 WIB - 12.00 WIB"
    console.log(jam)
    }

    javascript() //tidak ada argument
    //output : 10.00 WIB - 12.00 WIB
